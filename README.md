# home-media-server


## `kubernetes`
media-server on top of a Kubernetes.
It can be installed either on any k8s cluster or on an home server like Raspberry Pi with k3s. 

Includes the following pods:
- Jackett
- Sonarr
- Radarr
- Transmission
- Plex

## `Docker`
Media server on a simple container setup.
With an internal name-server and reverse proxy for easy access via sub-domains.


Includes the following applications:
- Jackett 
- Sonarr
- Radarr
- Transmission
- Plex
- Traefik
- Portainer
- dnsmasq
